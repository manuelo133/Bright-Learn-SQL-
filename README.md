This repository contains an end-to-end data analytics case study conducted on historical transactional data from Bright Coffee Shop.
The objective of the analysis is to generate actionable business insights to support a newly appointed CEO in improving revenue growth, product performance, and operational decision-making.

The project follows a modern analytics workflow, covering data architecture planning, SQL-based data processing, exploratory analysis, visualization, and executive-level reporting.


#### Business Objectives

The analysis aims to answer the following key business questions:

1. Which products and categories generate the highest revenue?
2. What time intervals drive peak sales performance?
3. How do sales trends vary across time and product types?
4. Which products underperform and require strategic intervention?
5. What data-driven recommendations can improve overall sales performance?
   

#### Data Architecture & Workflow

The project was designed using a structured data flow architecture, ensuring clarity, scalability, and reproducibility.

High-level flow:

1. Data Source

   Raw transactional sales data provided in Excel format

2. Planning & Architecture

   Data Flow Architecture Diagram designed using Miro
   Defined ETL logic, key metrics, and analytical outputs upfront

3. Data Processing & Storage

   Data converted to CSV and loaded into Snowflake
   SQL used for data cleaning, transformation, and aggregation

4. Data Analysis & Visualization

    Processed data exported to Microsoft Excel
    Pivot tables and charts created for exploratory and descriptive analysis

5. Presentation & Reporting

   Executive-ready insights presented using Canva
   Clear storytelling focused on business impact and recommendations
